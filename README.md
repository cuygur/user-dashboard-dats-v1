# DATS Dashboard

Dashboard for users in the DATS network.
 


## Run Locally

Clone the project

```bash
  git clone https://github.com/cuygur/user-dashboard-dats-v1.git
```

Go to the project directory

```bash
  cd user-dashboard-dats-v1
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```
